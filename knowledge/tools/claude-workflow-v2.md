Universal Claude Code workflow plugin that provides 7 specialized AI agents, 6 skills, and 8 automation hooks. Automatically selects the right agent based on your task.

## Key Features

- **7 Subagents**: Code reviewer, debugger, security auditor, docs writer, refactorer, test architect, orchestrator
- **6 Skills**: Pre-built capabilities for common workflows
- **8 Automation Hooks**: Custom triggers and integrations
- **Automatic Agent Selection**: Claude uses the appropriate agent based on your request
- **Zero Configuration**: Works immediately after cloning the repo

## Usage

```bash
git clone https://github.com/CloudAI-X/claude-workflow-v2
claude --plugin-dir ./claude-workflow
```
