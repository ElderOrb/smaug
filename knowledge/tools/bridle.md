Unified configuration manager for AI coding assistants. Manage profiles, install skills/agents/commands, and switch configurations across Claude Code, OpenCode, Goose, and Amp.

## Key Features

- TUI / CLI config manager for agentic harnesses
- Opt-in markers for within your config
- Full CLI support with JSON output for custom scripting
- Helper commands and harness status legend
- Awesome horse statue
