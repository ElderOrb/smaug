Web3 React Boilerplate for building decentralized applications. Provides essential components for wallet integration, user data display, and blockchain interactions.

## Key Features

- Login with Wallet/WalletConnect
- Show User Balances
- Show User NFTs
- Show User Transactions
- Show Functioning Wallet
