Fast, powerful, git-native ticket tracking in a single bash script. Dependency graphs, priority levels, zero setup. Inspired by Joe Armstrong's minimal viable program philosophy.

## Key Features

- Git-backed issue tracking
- Dependency graph management
- Priority levels
- Single bash script (no SQLite or background daemons)
- Markdown files with YAML frontmatter
- IDE integration via file naming
