Create multiple isolated Claude Code variants with custom providers (Z.ai, MiniMax, OpenRouter, LiteLLM). Enables official multi-agent orchestration features that are normally disabled.

## Key Features

- Multiple provider support (Z.ai, MiniMax, OpenRouter, LiteLLM)
- Complete isolation between Claude Code instances
- Official multi-agent orchestration capabilities
- Simple setup with npx cc-mirror quick command
